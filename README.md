# 학교 웹크롤링 대시보드
프로젝트 소개
이 프로젝트는 학교 홈페이지에서 일정, 행사, 공지사항 데이터를 자동으로 수집하여
대시보드(dashboard/index.html) 형태로 보여줍니다.

주요 기능
- 학교 일정, 행사, 공지사항 자동 크롤링
- 다양한 게시판 명칭과 이벤트 키워드 인식
- HTML 대시보드(캘린더, 테이블) 자동 생성
- GitHub 또는 GitHub Pages를 통한 손쉬운 공유

사용 방법
1. dashboard/index.html 파일을 브라우저에서 열어 대시보드를 확인하세요.
2. 대시보드를 최신화하려면 크롤링 스크립트를 실행하여 index.html 파일을 다시 생성하세요.
3. 파일을 공유하거나 GitHub Pages를 이용해 웹사이트로 공개할 수 있습니다.

파일 구조
- dashboard/index.html: 대시보드 HTML 파일
- config.yaml: 크롤링 설정 파일
- board_crawler.py: 크롤링 로직
- html_generator.py: 대시보드 생성기
- database.py: 데이터 저장 및 조회

주의사항
- 공개 저장소에는 개인정보나 민감한 데이터를 업로드하지 마세요.
- 대시보드 파일은 누구나 볼 수 있습니다.
